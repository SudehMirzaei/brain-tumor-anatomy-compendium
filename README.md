# 🧠 Brain Tumor Anatomy Compendium

> A comprehensive anatomy and radiology reference for understanding brain tumors from both a clinical and deep learning perspective.

---

## 📖 Overview

This repository provides structured documentation on the three most common primary brain tumors used in medical imaging datasets:

- **Glioma**
- **Meningioma**
- **Pituitary Tumor**

It also includes an introduction to MRI sequences commonly used in neuroimaging and a curated list of references. The content is designed for students, AI engineers, medical imaging researchers, and anyone interested in brain tumor anatomy and MRI interpretation.
